# HydraPet Smart Coaster

HydraPet Smart Coaster is a small hardware design project for a smart pet water coaster concept.

HydraPet Smart Coaster 是一个智能宠物饮水杯垫概念的小型硬件设计项目。

## Files

- `hydrapet_smart_coaster.scad` - OpenSCAD model source. / OpenSCAD 模型源文件。
- `hydrapet_smart_coaster_preview.html` - interactive preview page. / 交互式预览页面。
- `droplet-oled-preview.html` - OLED droplet preview. / OLED 水滴效果预览。
- `droplet-oled-variants.html` - OLED droplet visual variants. / OLED 水滴视觉方案变体。

## Getting Started

Open the HTML preview files in a browser, or load the `.scad` file in OpenSCAD to inspect and export the model.

你可以直接用浏览器打开 HTML 预览文件，也可以在 OpenSCAD 中加载 `.scad` 文件来查看和导出模型。
